# Snip

Capture any region of your screen and copy it to the clipboard in one keystroke.

## Usage

Press the configured hotkey (default **Ctrl+Alt+S**). Your screen dims and the cursor changes to a crosshair. Click and drag to draw a selection rectangle — the selected area is highlighted while the rest stays dimmed. Release the mouse to confirm the capture. Press **Escape** to cancel.

The captured image is placed on the clipboard and ready to paste anywhere.

## Actions

| Action | Default Hotkey | Description |
|---|---|---|
| Capture Region | Ctrl+Alt+S | Dim the screen and drag to select a region |

## Notes

- The hotkey can be rebound from the Actions tab on this page.
- Captures all connected monitors; you can select across monitor boundaries.
